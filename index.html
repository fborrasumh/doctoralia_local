Bienvenido a Doctoralia Local

Genera artículos de revisión científica usando modelos de IA que corren 100% en tu ordenador via Ollama. Sin API keys, sin costes, sin enviar datos a la nube.

🖥️
Detecta tu hardware
Usamos canirun.ai para saber qué modelos puede ejecutar tu máquina según RAM y VRAM
📦
Descarga el modelo
Un comando ollama pull descarga el modelo. Tarda unos minutos la primera vez.
🔒
100% local y privado
Todo corre en tu máquina. Tus datos, tu investigación, nunca salen de tu ordenador.
⚙️

Instala Ollama primero

Ollama es el servidor local que gestiona los modelos de IA. Necesitas tenerlo instalado y en ejecución.

1️⃣ Descarga Ollama desde ollama.com
2️⃣ Instálalo y ejecútalo (queda en background)
3️⃣ Elige un modelo en el panel de configuración
4️⃣ Copia el comando ollama pull <modelo> y ejecútalo en tu terminal
5️⃣ ¡Listo! El botón Probar conexión confirmará que todo funciona
🚀

Listo para empezar

Abre el HTML con un servidor local (no con file://) para que las APIs académicas funcionen correctamente.

💡 Servidor local rápido:
python3 -m http.server 8080
Luego abre http://localhost:8080

⚠️ Ollama debe estar corriendo en localhost:11434
